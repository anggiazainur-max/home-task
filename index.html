☁️ Konfigurasi Supabase Cloud Sync

Hubungkan aplikasi ke **Supabase** secara aman agar data tugas, pengingat, dan jadwal Anda otomatis tersinkronisasi di HP, Laptop, & anggota keluarga lainnya.
🛠 Cara Setup Database Gratis di Supabase:
  1. Daftar di supabase.com.
  2. Buat Project baru (pilih Singapura sebagai lokasi terdekat).
  3. Masuk ke menu **SQL Editor** di panel kiri -> klik **New Query**.
  4. Paste kode SQL di bawah ini, lalu klik tombol **Run**:
create table household_sync ( id text primary key, tasks jsonb default '[]'::jsonb, reminders jsonb default '[]'::jsonb, updated_at timestamp with time zone default timezone('utc'::text, now()) not null ); -- Matikan RLS agar client dapat membaca/menulis data secara langsung alter table household_sync disable row level security;
  1. Pergi ke **Project Settings (⚙️)** -> **API**.
  2. Salin **Project URL** dan **anon public key**, lalu masukkan ke kolom di atas!
