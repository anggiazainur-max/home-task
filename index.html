☁️ Konfigurasi Supabase Cloud Sync

Hubungkan aplikasi ke **Supabase** secara aman agar data tugas, pengingat, dan jadwal Anda otomatis tersinkronisasi di HP, Laptop, & anggota keluarga lainnya.
🛠 Cara Setup/Upgrade Database di Supabase:
  1. Masuk ke menu **SQL Editor** di panel kiri dashboard Supabase -> klik **New Query**.
  2. Paste kode SQL di bawah ini, lalu klik tombol **Run**:
create table if not exists household_sync ( id text primary key, tasks jsonb default '[]'::jsonb, reminders jsonb default '[]'::jsonb, members jsonb default '[]'::jsonb, updated_at timestamp with time zone default timezone('utc'::text, now()) not null ); -- JALANKAN INI JIKA TABEL SUDAH PERNAH DIBUAT SEBELUMNYA: alter table household_sync add column if not exists members jsonb default '[]'::jsonb; -- Matikan RLS agar client dapat membaca/menulis data secara langsung alter table household_sync disable row level security;
  1. Pergi ke **Project Settings (⚙️)** -> **API**.
  2. Salin **Project URL** dan **anon public key**, lalu masukkan ke kolom di atas!
